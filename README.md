📈 EconEp Paper Trading Simulator
A classroom-focused stock market simulator built for middle school economics education. This application allows students to track live market news and manage a virtual portfolio, teaching them the principles of long-term investing and market dynamics.

🚀 Features
Live Market Watch: Fetches real-time stock data for a selection of tickers using yfinance.

Student Portfolios: Secure login system to allow each student to track their own virtual cash and stock holdings.

Educational Focus: Designed for simplicity, helping students understand the relationship between asset price and total value.

🛠 Tech Stack
Framework: Streamlit

Authentication: streamlit-authenticator

Data Source: yfinance

Environment: Python 3.14

💡 How to run locally
Clone the repository to your computer.

Install dependencies: pip install -r requirements.txt

Run the app: streamlit run streamlit_app.py
